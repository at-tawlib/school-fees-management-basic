    App Description

    School Fees Tracker is a desktop application designed to help schools manage student fee payments efficiently. It allows tracking of billed fees, payments, arrears, and provides a summary of financial activities for different academic years and terms.

    Technologies Used

    • Electron
    • JavaScript (ES6+)
    • SQLite
    • HTML & CSS
